NOTE: In C#, the operator to divide is "/" and the operator to multiply is "*"!

Grade Calculator

Right, then let's make a fake little grade calculator. So, you give the program a percentage, and it will say whether that's an "A*, A, B, C, D etc."
These are the numbers we will be using:
91 - 100% = A*
81 - 90% = A
71 - 80% = B
61 - 70% = C
51 - 60% = D
Below 50% = E

Use integers - not floats for this one.

IMPORTANT TIP
You have to use "else if" on this one. Because if you say something like if (percentage > 90) and then later on say if (percentage > 80), then it will run both of them.
So, in that scenario, if you got "96%", the first if statement would run, because your percentage is higher than "91", it will run that first "if" statement, and your grade will be "A*", but, then because all the others will also run, that "A*" would be overwritten with an "D" by the if (percentage > 50).

However, if you did if (percentage > 90), followed by else if (percentage > 80), then it won't run that second if, if the percentage was above 91, so that doesn't happen.
So, use "else if"s.

Extension

The Ultimate Adder

So, in this, we create a variable called "total" - everytime the user enters another number, it will add to the thing.
Then, we, of course, actually ask them to input how many numbers they want to add together.
After that, we use a "for" loop in order to go enter a specific amount of number - so, if "numberOfNumbersToAdd" is "3", the code inside the for loop will run 3 times - if it's 4, the code inside the for loop will run 4 times.
Then, we ask them to enter that number, and we take the number they entered, and add it to the total - so, if total was "5", and they entered "3", it will become "8"
And, finally, we output the total at the end!
Oh, and, for the extension, we wrap the whole thing in a "while (true)", so it will never end unless you press the close button. 

                            

while (true) {

    int total;
    int numberOfNumbersToAdd;
    total = 0;

    Console.WriteLine("Enter the amount of numbers you want to add together.");
    numberOfNumbersToAdd = int.Parse(Console.ReadLine());

    for (int i = 0; i < numberOfNumbersToAdd; i++) {

        int enteredNumber;

        Console.WriteLine("Enter number " + (i + 1));
        enteredNumber = int.Parse(Console.ReadLine());

        total = total + enteredNumber;

    }

    Console.WriteLine("The final total is: " + total);
    Console.ReadLine();

}
                            
                            
// Make the code run infinitely.

}

The Mean Calculator

Exactly like the ultimate adder, except at the end we divide their number by how many numbers they gave.
Also, as a part of the extension, we're checking if they chose to process 0 items, in which case, we'll reject it. 

                            

while (true) {

    int total;
    int numberOfNumbersToAdd;
    float mean;
    total = 0;

    Console.WriteLine("Enter the amount of numbers you want to process.");
    numberOfNumbersToAdd = int.Parse(Console.ReadLine());

    if (numberOfNumbersToAdd == 0)
        Console.WriteLine("You can't process 0 numbers!");
    else {
        for (int i = 0; i < numberOfNumbersToAdd; i++) {

            int enteredNumber;

            Console.WriteLine("Enter number " + (i + 1));
            enteredNumber = int.Parse(Console.ReadLine());

            total = total + enteredNumber;

        }

        mean = total / numberOfNumbersToAdd;

        Console.WriteLine("The mean is: " + mean);
    }
    Console.ReadLine();

}

Standard Deviation

NOTE: This one is quite a challenge and quite a long one too, but definitely worth it!
One thing you can calculate in maths is something called the standard deviation, and it's a very long and time-consuming thing to calculate - so, we're going to write a program to calculate it for us!

So, we can take a list of any amount of numbers and calculate the standard deviation of them, what the standard deviation is, is essentially an average of how far away from the mean all of the items are, it gives us a good idea of if the data is "skewed" in a certain direction.

Alright, here's how to calculate it:

1. Take in all the numbers of course, as you can see in the console, you say how many items you want to enter, and just enter them.

2. Work out the mean of all of the numbers, how to work out the mean is below.

3. We need to get the sum of all of the items squared. So, go through every item, square them (by multiplying them by themselves), then add them to some sort of total - it's exactly like with the "mean", except you square each item before adding them to the total.

4. Take that total we just got and divide that by the total number of items, exactly like we would with the mean. Just like with the mean, put this into a double variable.

5. We take the number we just calculated and subtract it from the mean squared. So, it will look something like this: lastNumber - (mean * mean), put this into a double too

6. Finally, we take that number we just got and work out the square root of it, that's your final answer, the square root of that number. NOTE: To work out the square root, use "variable = Math.Sqrt(number)", putting the number you want to square root inside place of the "number", of course.
NOTE: "Math.Sqrt" gives back a double, and takes in a double, so put it into a "double" variable, and make sure to give it a double too.

How to work out the mean:
1. Create a "total" variable.
2. Take all of items and add them up, into some kind of "total" variable, to do that go through each item, and within the "for" loop, for each item, just add it to the total using "+=". This will need to go into a "double"!
3. Finally, take that "total" and divide it by the number of items there are.

What is a double?
A double is just a more accurate "float". Floats are 32-bits, while a double is 64-bits, so you can get more decimal places in a double

If you've done something wrong - you will probably get "NaN" as your final answer, check through your code carefully.
To check if you did it right, just enter in some numbers into the console, and enter the same numbers into your program, and you should get the same result!

// Ask the user how many numbers they want to use to calculate the standard deviation.
Console.WriteLine("Choose how many numbers you want to work out the standard deviation for.");
int numberOfItems = int.Parse(Console.ReadLine());

if (numberOfItems == 0)
    Console.WriteLine("You can't have 0 items!");
else {

    // Create an array based on that size.
    int[] items = new int[numberOfItems];

    // Ask the user to input all of those numbers.
    Console.WriteLine("Input all the numbers you want to work out the standard deviation for:");
    for (int i = 0; i < numbersOfItems; i++) {
        Console.WriteLine("Input Number " + (i + 1));
        items[i] = int.Parse(Console.ReadLine());
    }

    // Work out the mean.
    int total = 0;
    for (int i = 0; i < numberOfItems; i++)
        total += items[i];

    double mean = total / numberOfItems;

    // Work out the sum of all of the items squared. Then, divide that sum by the numbers of items.
    int squaredSum = 0;
    for (int i = 0; i < numberOfItems; i++)
        squaredSum += items[i] * items[i];

        double dividedSquareSum = squaredSum / numberOfItems;

    // Do the final calculation, and work out the square root of that.
    double beforeSquareRoot = dividedSquareSum - (mean * mean);
    double finalResult = Math.Sqrt(beforeSquareRoot);

    // Finally, print out the results - we'll also print out the mean for good measure :).
    Console.WriteLine("Mean: " + mean);
    Console.WriteLine("Standard Deviation: " + finalResult);
}
